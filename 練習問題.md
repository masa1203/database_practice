## 4.3 テーブルから集計したデータを別テーブルにインサートする

``` SQL
create table ShohinSaeki(
    shohin_id CHAR(4) NOT NULL,
    shohin_mei VARCHAR(100) NOT NULL,
    hanbai_tanka INTEGER  ,
    shiire_tanka INTEGER  ,
    saeki INTEGER  ,

insert into ShohinSaeki
```

## 5.4 相関サブクエリをつかったビュー

``` sql
select shohin_id,
```
